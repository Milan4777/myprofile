# Milan Shammi — Portfolio (scaffold)

This is a minimal HTML/CSS portfolio scaffold. Update the placeholders in `index.html` with your details, add images to `assets/`, and edit `styles.css` for colors and typography.

Preview locally:

```bash
# on Windows PowerShell
Start-Process index.html

# or run a simple HTTP server (Python 3)
python -m http.server 8000
# then open http://127.0.0.1:8000
```
